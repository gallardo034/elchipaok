# El Chipa OK

Landing estática para el emprendimiento de chipá congelado El Chipa OK.

## Archivos

- `index.html`: estructura completa de la landing.
- `styles.css`: identidad visual responsive.
- `script.js`: navegación mobile y estado del header.
- `assets/chipa-hero.png`: imagen hero provisional generada para el sitio.

## Ver localmente

Abrir `index.html` directamente en el navegador.

## Publicación en Vercel

Importar este repositorio como proyecto estático. No requiere build command ni framework.

Configuración recomendada:

- Framework Preset: Other
- Root Directory: `./`
- Build Command: vacío
- Output Directory: vacío
- Install Command: vacío

## Notas para versión final

Cuando estén disponibles el logo y las fotos reales, reemplazar o sumar esos archivos dentro de `assets/` y ajustar las rutas en `index.html`. La sección de retiro ya incluye el mapa embebido de Google Maps.
